# Multi-Module Dashboard Application

A comprehensive, modern web application showcasing multiple modules and advanced features. This project demonstrates full-stack development skills with React, TypeScript, and modern web technologies.

## 🚀 Features

### Core Modules
- **📊 Overview Dashboard** - Main dashboard with key metrics and module navigation
- **✅ Task Management** - Interactive task creation, editing, and tracking
- **💬 Real-time Chat** - Team communication with message history
- **🛒 E-commerce** - Product management and order processing
- **🌤️ Weather** - Real-time weather information and forecasts
- **📈 Analytics** - Data visualization and business insights
- **💪 Fitness Tracker** - Workout logging and health metrics
- **👨‍🍳 Recipe Manager** - Recipe collection and meal planning
- **🧠 AI Insights** - Intelligent recommendations and analytics

### Technical Features
- **Modern React 18** with TypeScript for type safety
- **Responsive Design** that works on all devices
- **Real-time Updates** with WebSocket integration
- **Advanced Animations** using Framer Motion
- **State Management** with Zustand
- **Form Handling** with React Hook Form
- **Data Visualization** with Recharts
- **API Integration** with Axios
- **Modern UI/UX** with Tailwind CSS

## 🛠️ Tech Stack

### Frontend
- **React 18** - Modern React with hooks and concurrent features
- **TypeScript** - Full type safety and better development experience
- **Vite** - Fast build tool and development server
- **Tailwind CSS** - Utility-first CSS framework
- **Framer Motion** - Smooth animations and transitions

### Libraries & Tools
- **React Router** - Client-side routing
- **Zustand** - Lightweight state management
- **React Hook Form** - Performant forms with validation
- **Recharts** - Composable charting library
- **Socket.io Client** - Real-time communication
- **Axios** - HTTP client for API calls
- **React Hot Toast** - Toast notifications
- **Lucide React** - Beautiful icons

### Development Tools
- **ESLint** - Code linting and formatting
- **PostCSS** - CSS processing
- **Autoprefixer** - CSS vendor prefixing

## 📦 Installation

1. Clone the repository:
```bash
git clone <your-repo-url>
cd multi-dashboard
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open [http://localhost:3001](http://localhost:3001) in your browser.
